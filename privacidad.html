Política de Privacidad

Cómo manejamos la información cuando usas ChequealoAI.

Última actualización: junio 2025

1. Información que procesamos

Cuando verificas una noticia en ChequealoAI, el texto que ingresas es enviado temporalmente a nuestros servidores para ser procesado. No almacenamos este texto de forma permanente en servidores propios.

  • Consultas de verificación: el texto ingresado se envía a la API de OpenAI y a la API de Serper para generar el análisis. Estas empresas tienen sus propias políticas de privacidad.
  • Historial local: el historial de tus verificaciones se guarda exclusivamente en el almacenamiento local de tu navegador (localStorage). No lo vemos ni accedemos a él desde nuestros servidores.

2. Uso de terceros

ChequealoAI utiliza los siguientes servicios externos para funcionar:

  • OpenAI (api.openai.com): recibe el texto de tu consulta para generar el análisis con IA. Consulta su política de privacidad.
  • Serper (serper.dev): recibe tu consulta para buscar noticias relacionadas en tiempo real. Consulta su política de privacidad.
  • Netlify: plataforma de hosting que puede registrar logs de acceso estándar (IP, fecha, URL solicitada). Consulta la política de privacidad de Netlify.

3. Cookies y seguimiento

ChequealoAI usa Google Analytics para medir métricas agregadas de uso (páginas vistas e interacción general). No utilizamos estos datos para identificarte personalmente. Además, usamos localStorage del navegador para guardar tu historial de verificaciones de forma local.

4. Menores de edad

ChequealoAI es un servicio de acceso público. No recopilamos intencionalmente información personal de ningún usuario, incluyendo menores de edad.

5. Cambios a esta política

Podemos actualizar esta política ocasionalmente. Los cambios se publicarán en esta misma página con una nueva fecha de actualización.

6. Contacto

Si tienes preguntas sobre esta política, puedes contactarnos a través de la página Acerca de.
