# Release Notes

## v1.0.0 - Primer Release

Fecha: 2026-06-17

### ✨ Incluye
- Interfaz web para validar titulares o textos de noticias.
- Veredicto automático: **Real**, **Dudosa** o **Falsa**.
- Resumen y razones de validación en lenguaje natural.
- Fuentes sugeridas para verificar la información.
- Integración segura con OpenAI mediante Netlify Function.

### 🔐 Seguridad y despliegue
- Uso de `OPENAI_API_KEY` como variable de entorno en servidor.
- Configuración lista para despliegue en Netlify con funciones serverless.
